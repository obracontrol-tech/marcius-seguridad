📊 Configurar Google Sheets
Paso 1 · Abre sheets.google.com y crea una hoja nueva.
Paso 2 · Ve a Extensiones → Apps Script
Paso 3 · Pega este código:
function doPost(e){
  var datos = JSON.parse(e.postData.contents);
  var hoja = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  hoja.appendRow([
    datos.fecha, datos.hora, datos.agente, datos.puesto,
    datos.tipo, datos.descripcion, datos.lat, datos.lon,
    datos.rondas, datos.incidencias
  ]);
  return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON);
}
